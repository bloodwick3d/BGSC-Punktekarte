# ⛳ BGSC Punktekarte - Version 1.0.0

Eine moderne, intuitive Android-App für Minigolf-Spieler des **BGSC "Gut Schlag" Gladbeck e.V.** und alle begeisterten Minigolfer. Diese App ersetzt die klassische Papier-Punktekarte durch eine digitale, leistungsstarke Lösung.

---

## 🌟 Offizieller Release 1.0.0
Dies ist die erste stabile Version der BGSC Punktekarte. Sie wurde von Grund auf mit **Jetpack Compose** entwickelt, um eine flüssige und moderne Benutzererfahrung zu bieten.

## 🚀 Kern-Funktionen

### 1. Intelligente Scorekarte
*   **Intuitive Eingabe:** Schnelle Punkteerfassung über das innovative Circle-Menu.
*   **Live-Statistiken:** Automatische Berechnung von Gesamtergebnis, Rundenschnitten und Farbkennzeichnung nach offiziellen System-Standards.
*   **Multi-Player:** Unterstützung für bis zu 10 Spieler gleichzeitig.
*   **Multi-Runden:** Verwalte bis zu 4 Runden in einem Spiel.

### 2. Turnier-Modus (Pro-Features)
Speziell für Turnierspieler entwickelt:
*   **Bahnen-Notizen:** Speichere Ball-Empfehlungen, Abschlagspunkte und taktische Hinweise für jede der 18 Bahnen.
*   **System-Support:** Spezifische Einstellungen für Eternit, Beton, Filz, Cobi und Sterngolf.
*   **Export/Import:** Teile deine Turniernotizen mit Teamkollegen über verschlüsselte GZIP-Dateien.

### 3. Analyse & Teilen
*   **Spiel-Historie:** Alle Spiele werden lokal gespeichert und können jederzeit eingesehen oder gelöscht werden.
*   **Ergebnis-Sharing:** Generiere hochauflösende Ergebniskarten als Bild, inklusive Vereins-Logo und professionellem Layout, um sie direkt in WhatsApp oder Social Media zu teilen.

## 🛠️ Installation

1.  Gehe zum Reiter [**Releases**](https://github.com/bloodwick3d/BGSC-Punktekarte/releases).
2.  Lade die `BGSC Punktekarte v1.0.0.apk` herunter.
3.  Öffne die Datei auf deinem Android-Smartphone.
4.  Erlaube ggf. die Installation aus "unbekannten Quellen".

---

## 💻 Entwicklung & Build

Das Projekt ist Open-Source für Vereinsmitglieder und nutzt moderne Android-Technologien:
*   **Sprache:** Kotlin
*   **UI:** Jetpack Compose mit Material 3
*   **Datenbank:** Room (SQLite)
*   **CI/CD:** GitHub Actions für automatische Releases

### Build-Befehl
```bash
./gradlew assembleRelease
```
